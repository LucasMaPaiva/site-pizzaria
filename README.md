# 🍕 Dois90 Experiência Gastronômica

> Onde a tradição encontra o contemporâneo. Desde 1998, celebrando os bons momentos da vida com sabor e qualidade.

Este projeto é o site oficial da **Dois90**, apresentando nossa história, cardápio multitemático (Pizzaria, Cafeteria, Gelateria e Padaria) e nossas unidades em Boa Vista - RR.

## 🚀 Como Executar Localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

### Pré-requisitos
- [Node.js](https://nodejs.org/) (recomendado v18+)
- Gerenciador de pacotes (npm ou yarn)

### Passo a Passo

1. **Clonar o repositório:**
   ```bash
   git clone <url-do-repositorio>
   cd site-pizzaria
   ```

2. **Instalar dependências:**
   ```bash
   npm install
   ```

3. **Configurar variáveis de ambiente:**
   Crie um arquivo `.env.local` na raiz do projeto e adicione sua chave do Gemini:
   ```env
   VITE_GEMINI_API_KEY=sua_chave_aqui
   ```

4. **Iniciar o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
   Acesse: `http://localhost:3000`
